# WorkBook_MonthlyRgCostDiff
リソース グループ毎に先月と今月の月額料金とその差額を表示するブック テンプレートです。
通貨の単位は契約などにより異なりますが、このテンプレートでは $ を表示するように設定しています。
お使いの環境にあわせて、列の表示設定を変更してください。
